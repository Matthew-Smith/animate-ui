## Usage

### Global

```tsx
<CursorProvider>
  <Cursor>
    <div>Cursor</div>
  </Cursor>
  <CursorFollow>
    <div>Cursor Follow</div>
  </CursorFollow>
</CursorProvider>
```

### Local

```tsx
<CursorProvider>
  <CursorContainer>
    <Cursor>
      <div>Cursor</div>
    </Cursor>
    <CursorFollow>
      <div>Cursor Follow</div>
    </CursorFollow>
  </CursorContainer>
</CursorProvider>
```
